# 测试样本说明

## 样本文件

```txt
samples/demo-sales.csv
```

该样本用于验证 ReportPilot MVP 闭环：

```txt
上传 CSV
→ FileAsset 入库
→ ImportJob 入库
→ worker 解析 CSV
→ 字段类型识别
→ 数据质量检查
→ DeepSeek 生成报告
→ Report 入库
→ 前端展示报告
```

## 字段说明

| 字段 | 说明 | 预期类型 |
| --- | --- | --- |
| order_id | 订单编号 | text |
| order_date | 下单日期 | date |
| region | 地区 | category |
| channel | 渠道 | category |
| product | 商品 | category |
| customer_type | 客户类型 | category |
| quantity | 数量 | number |
| unit_price | 单价 | number |
| revenue | 销售额 | number |
| discount_rate | 折扣率 | number |
| payment_status | 支付状态 | category |
| note | 备注 | text |

## 预期检查结果

- 能识别数字、日期、分类和文本字段。
- 能生成字段画像、行数、列数和数据质量摘要。
- 能发现少量空值和重复记录。
- 能基于真实字段生成推荐图表。
- 报告详情页不展示原始 AI JSON。

## 验证步骤

1. 启动 Web：

```bash
pnpm dev
```

2. 启动 worker：

```bash
pnpm worker:imports
```

3. 注册并登录。
4. 上传 `samples/demo-sales.csv`。
5. 等待任务处理完成。
6. 查看文件详情和报告详情。

## 注意事项

- 不要把真实敏感数据放入测试样本。
- 上传目录为 `.local-storage/uploads`，不会提交到 Git。
- DeepSeek 只接收脱敏样本、字段画像、质量摘要和基础统计，不接收完整 CSV。
